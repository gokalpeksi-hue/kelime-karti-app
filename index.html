Google Drive ile senkron
Google bağlantısı (Drive yedekleme için gerekli)

Client ID nasıl alınır? (yaklaşık 5 dk, ücretsiz)
  1. console.cloud.google.com adresine Google hesabınla gir ve yeni bir proje oluştur (örn. MyDictionary).
  2. Soldaki menüden APIs & Services → OAuth consent screen: External seç, uygulama adını yaz, e-postanı seç, kaydet. Son adımda Publish app düğmesine bas.
  3. APIs & Services → Library bölümünden Google Drive API'yi bulup Enable et.
  4. APIs & Services → Credentials → + Create Credentials → OAuth client ID → Web application seç.
  5. Authorized JavaScript origins alanına şunu ekle: https://gokalpeksi-hue.github.io
  6. Oluşan Client ID'yi kopyala, yukarıdaki kutuya yapıştır ve Bağlantıyı kaydet'e bas. Artık "Drive'a yedekle" ve "Drive'dan yükle" düğmeleri çalışır.
